**Mб»Ҙc tiГӘu**: TбәЎo module kб»№ nДғng bбәЈo mбәӯt cГі thб»ғ tГЎi sб»ӯ dб»Ҙng

**Chб»©c nДғng chГӯnh**:

| Function | MГҙ tбәЈ |
|----------|-------|

```

рҹҹЎ MEDIUM (40-69): CAUTION - Thбәӯn trб»Қng  
рҹҹў LOW (0-39): SAFE - An toГ n
```

**Sб»ӯ dб»Ҙng**:
```javascript

// PhГўn tГӯch contract

// QuГ©t token

if (tokenRisk.riskScore > 70) {
  console.log("вҡ пёҸ Token nguy hiб»ғm!");

```
